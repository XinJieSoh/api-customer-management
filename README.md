# API Customer Information Management

## Project setup
Git clone the project.

Copy the example.env file and rename it to the .env, change the CONNECTION_STRING to your own MongoDB connection string.

Install the packages.

```sh
npm install
```

To run it:

```sh
npm run dev
```
